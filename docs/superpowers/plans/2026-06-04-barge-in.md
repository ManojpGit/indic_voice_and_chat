# Barge-in (dev console) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let the user interrupt the agent mid-utterance: the browser detects user speech during agent playback, stops playback instantly, and signals the server to cancel the in-flight turn so the agent stops and the interruption becomes the next turn.

**Architecture:** Browser-side detection (VAD on the echo-cancelled mic) + instant local `stopPlayback()` + a `{"type":"barge_in"}` signal. The server's transport-agnostic `_handle_barge_in()` sets the in-flight turn's `cancel_event` and clears the busy gate; the engine already stops the LLM loop + TTS on cancel; `handle_turn_text` records the user turn, drops the abandoned agent reply, and returns to LISTENING. The cancel/turn-handling core is shared so telephony (server-side detection) reuses it later.

**Tech Stack:** Python 3.12, asyncio, FastAPI WebSocket, vanilla JS (Web Audio + AudioWorklet), pytest + pytest-asyncio. Spec: `docs/superpowers/specs/2026-06-04-barge-in-design.md`.

```javascript
      workletNode.port.onmessage = (e) => {
        if (!running) return;
        const stillPlaying = playCursor > audioCtx.currentTime + 0.05;
        const speaking = agentSpeaking || stillPlaying;
        if ($("halfDuplex").checked && speaking) {
          // Agent is speaking. With barge-in off, drop the frame (half-duplex).
          if (!$("bargeIn").checked) { bargeMs = 0; return; }
          // Barge-in on: VAD the echo-cancelled mic for sustained user speech.
          let sum = 0;
          for (let i = 0; i < e.data.length; i++) sum += e.data[i] * e.data[i];
          const rms = Math.sqrt(sum / e.data.length);
          const frameMs = (e.data.length / audioCtx.sampleRate) * 1000;
          bargeMs = rms > BARGE_RMS ? bargeMs + frameMs : 0;
          if (bargeMs < BARGE_MS) return;          // not enough sustained speech yet
          // Confirmed interruption: stop the agent instantly + tell the server.
          bargeMs = 0;
          stopPlayback();
          agentSpeaking = false;
          if (ws.readyState === WebSocket.OPEN) ws.send(JSON.stringify({ type: "barge_in" }));
          // fall through → stream this frame and the rest of the interruption
        } else {
          bargeMs = 0;
        }
        const int16 = downsampleToInt16(e.data, audioCtx.sampleRate);
        if (ws.readyState === WebSocket.OPEN) ws.send(int16.buffer);
      };
```
